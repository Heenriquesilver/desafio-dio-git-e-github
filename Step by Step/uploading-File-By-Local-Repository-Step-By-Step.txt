1 - criado meu repositorio remotamente, irei clonar o mesmo dando um gitBashHere na pasta criada localmente e passando o comando .git clone linkDoRepositorio.

2- já com os arquivos na minha maquina e feitas as altereçãoes localmente, para subir o mesmo novamente para a nuvem primeiro vamos mover o arquivo para staged com o comando git add .

3- em seguida realizamos o commit com o comando git commit -m "mensagem passada no commit".

4-quase pronto agora nosso arquivo já está pronto para subir, bastando dar um git push origin main. atenção, este origin main é o nome do diretorio podendo variar conforme alguma edição ou da nossa escolha.gi
